How to Simulate Fetching from Server

    Steps
  • First of all, be sure you have nodejs and npm installed on your computer.
  • Then using your terminal please install json-server globally: 
    sudo npm install json-server -g
  • After that go to the json-data subfolder (assuming you're in the demo folder):
    cd json-data
  • Run the json-server using following command: 
    json-server --watch db.json
  • Now you can click on the bellow button to simulate fetching notifications from the server:
    Fetch now!
  • After simulation you can terminate the json-server by pressing CTRL+C in the termial window where you ran the server.
